## Descripción del proyecto

Este proyecto consiste en el desarrollo de un prototipo funcional (MVP) de un tablero comunitario de alertas orientado a comunidades rurales del departamento del Meta, Colombia. La solución permite reportar y visualizar incidentes de seguridad comunitaria mediante herramientas gratuitas y accesibles.

El sistema busca fortalecer la participación ciudadana, mejorar la comunicación vecinal y facilitar la toma de decisiones comunitarias relacionadas con la seguridad.

---

## Objetivo general

Desarrollar un prototipo MVP de un tablero comunitario de alertas para reportar y visualizar incidentes de seguridad, facilitando la cooperación vecinal y la toma de decisiones en la zona.

---

## Funcionalidades principales

- Registro anónimo de incidentes.
- Visualización geográfica de alertas.
- Actualización automática de reportes.
- Consulta de incidentes recientes.
- Acceso desde dispositivos móviles y navegadores web.

---

## Tecnologías utilizadas

- Google Forms
- Google Sheets
- Google My Maps
- GitHub Pages
- HTML5

---

## Arquitectura del sistema

Usuario → Google Forms → Google Sheets → Google My Maps → Visualización web

---

## Metodología de desarrollo

El proyecto fue desarrollado utilizando la metodología ágil SCRUM, organizando el trabajo mediante sprint de análisis, diseño, desarrollo y validación del MVP.

---

## Nivel de maduración tecnológica

El prototipo desarrollado alcanza un nivel TRL5, debido a que fue validado mediante pruebas funcionales y de usabilidad en un entorno simulado con usuarios de prueba.
